# Stage 1: 데이터 수집 및 표준화 (Ingestion)

## 개요
다양한 소스(iRacing API, MoTeC CSV)에서 수집된 비정형 텔레메트리 데이터를 표준화된 형식으로 변환합니다.

## 주요 기능
- **헤더 자동 탐지**: Time 필드를 기반으로 CSV 헤더 위치 자동 탐지
- **단위 변환**: mph → kph, m/s → km/h 자동 변환
- **필드 정규화**: snake_case로 필드명 통일
- **중복 제거**: 세션 레벨 SHA-256 해시 기반 중복 검사

## 성과
- 파싱 성공률 **95% 이상** 달성
- 수동 전처리 작업 **100% 제거**
