# Stage 2: 고성능 저장소 구축 (Storage)

## 개요
정제된 텔레메트리 데이터를 PostgreSQL에 효율적으로 저장합니다.

## 주요 기능
- **Bulk Insert**: 500개씩 배치 처리로 대용량 데이터 적재 최적화
- **3NF 스키마 설계**: 정규화된 관계형 스키마로 데이터 무결성 보장
- **파티셔닝 & 인덱싱**: 대용량 데이터 조회 성능 최적화
- **Foreign Key 제약조건**: 데이터 무결성 보장

## 성과
- 500개 세션 처리 시간: **30분 이내** (세션당 약 3.6초)
- 데이터 일관성: **100%** (중복 레코드 0건)
